RadReveal

RadReveal is a JavaScript library to help you extend Reveal.js. Take a look!

Download this project as a .zip file Download this project as a tar.gz file

Reveal.js is a great way to create a slide deck!

But do you find yourself wishing it could do more? Or always making the same hacks to the code?

If so, RadReveal may be for you! Watch the demo slides or check out the project readme to find out how you can create or use RadReveal add-ons!
